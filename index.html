Tổng quan

Số hóa chấm thi thăng đai — linh hoạt mọi điều kiện mạng

Hệ thống phục vụ chấm thi thăng đai võ thuật theo mô hình điện tử trên cùng một bộ mã nguồn, hỗ trợ hai kịch bản vận hành:

  • Online 3 miền — giám khảo Bắc / Trung / Nam chấm trực tiếp, dữ liệu tập trung, Scoreboard realtime trên LED.
  • Offline cục bộ — chấm tại một CLB khi mất mạng; lưu tạm trên thiết bị, đồng bộ Cloud sau khi có Internet.

Người dùng truy cập qua trình duyệt trên Mobile / Tablet / Laptop. Frontend host trên GitHub Pages, ưu tiên PWA (Offline First) và responsive.

Backend / API

Switch Mode — một bộ code, hai hướng xử lý

Mode 1 · Online 3 miền

Javascript gọi API Apps Script qua Internet → ghi Google Sheets realtime.

Mode 2 · Offline cục bộ

Javascript ghi/đọc localStorage trên thiết bị → đồng bộ Sheets/Drive sau thi.

Chế độ hoạt động

Offline cục bộ & Online 3 miền

Online · Live Multi-Region

Online 3 miền

  • Internet liên tục (4G/Wi-Fi) tại mọi điểm cầu
  • Nút GỬI ĐIỂM → Fetch POST → Apps Script → Sheets ngay
  • Dashboard & Scoreboard cập nhật realtime
  • Hạ tầng video Meet / Zoom / Teams (kịch bản Live)

Đồng bộ ngay khi gửi điểm — hậu kỳ chủ yếu xuất PDF và lưu Drive.

Offline · 1 CLB

Offline cục bộ

  • PWA cache giao diện — không bắt buộc Internet khi thi
  • Nút LƯU ĐIỂM → ghi localStorage
  • Phương án A: Hotspot 4G → hoạt động gần như Online
  • Phương án B: Mất sóng 100% → xuất QR, Thư ký quét tổng hợp

Sau thi: nút ĐỒNG BỘ đẩy dữ liệu lên Google Sheets & Drive.

So sánh Offline vs Online

Tiêu chí Offline (1 CLB) Online (3 miền)
Internet khi thi Không bắt buộc (phương án B) Bắt buộc, liên tục
Lưu trữ tạm localStorage trên thiết bị Gửi trực tiếp qua API
Nút trên app LƯU ĐIỂM GỬI ĐIỂM
Đồng bộ Cloud 1 lượt sau thi (khi có mạng) Realtime khi gửi điểm
Dashboard / Scoreboard Không realtime (B); gần như có (A qua 4G) Có, realtime trên LED 3 miền
Phạm vi 1 CLB / cục bộ 3 miền + Tổ đình

Setup hạ tầng hình ảnh

Polling GET mỗi 5–10 giây. Bảng xếp hạng tự cập nhật trên LED — vinh danh Thủ khoa ngay lập tức.

Vai trò người dùng

Phân công trong kỳ thi

Giám khảo

Tại điểm cầu / CLB

  1. Mở Web App trên GitHub Pages (PWA)
  2. Xem thi & nhập điểm, nhận xét
  3. Online: GỬI ĐIỂM · Offline: LƯU ĐIỂM

Thư ký Online

Tại trung tâm / Tổ đình

  1. Theo dõi Dashboard (GET API từ Sheets)
  2. Rà soát, đối chiếu điểm
  3. Nhấn Khóa phiếu → PENDING_APPROVAL
  4. Xuất PDF, in USB, hỗ trợ quét QR (Offline B)
Chánh chủ khảo Online

Tại trung tâm / Tổ đình

  1. Xem bảng điểm & nhận xét trên Web App
  2. Đối chiếu với hình ảnh camera (Live)
  3. Nhập mã PIN để duyệt
  4. Trạng thái → OFFICIALLY_APPROVED
